# PSPDFKit for iOS - Instant iOS Example

This is a small example app demonstrating [PSPDFKit for iOS](https://pspdfkit.com/pdf-sdk/ios/) and [PSPDFKit Instant](https://pspdfkit.com/instant/). This example is the iOS client counterpart of https://github.com/PSPDFKit/pspdfkit-server-example-rails.

## Requirements

* The latest [Xcode](https://developer.apple.com/xcode/).
* A PSPDFKit Document Engine instance.

## Getting Started

* Start a local PSPDFKit Document Engine instance. See the [Server Example Projects](https://pspdfkit.com/guides/server/current/pspdfkit-server/example-projects/) guide to get your instance up and running.
* Click on the green "Code" button and select the "Open with Xcode" option.<sup>[1](#footnote1)</sup>
* Follow the steps in Xcode to clone and open the project.
* Build and run.
